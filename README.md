# PBSAS Finance Operations Database

Internal finance and operational CRM system for:

* PRO-BOXING SPORTS AGENCY AND SERVICES LLC (PBSAS)
* UAE Athletic Commission
* WBC Eurasia

## Purpose

This system is being developed to manage:

* Expenses
* Revenues
* Reimbursements
* Events
* Officials
* Licensing records
* Fighters
* Coaches
* Promoters
* Commission operations
* Receipt storage
* Financial reporting

## Planned Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Supabase
* PostgreSQL
* Vercel

## Current Development Phase
